# Greek Verb Master 🇬🇷

**Greek Verb Master** is a lightweight, browser-based web app designed to help learners practice **Modern Greek verbs** through interactive exercises and a built-in verb dictionary.

The app runs entirely on the client using **Vue 3**, **Tailwind CSS**, and a local `conjugation.json` data file — no backend required.

## ✨ Features

Greek Verb Master includes four interactive learning modes:

### 1. 🧠 Meaning Matcher

- Match Greek verbs with their English meanings
- Practice in both directions: **Greek → English** and **English → Greek**
- Multiple-choice questions

### 2. ✍️ Conjugation Drill

- Practice specific verb conjugations
- Translate individual verb forms
- Supports **Greek ↔ English** directions

### 3. 🧩 Table Puzzle

- Drag-and-drop conjugation game
- Match verb forms with the correct pronouns
- Random verb and tense selection for variety

### 4. 📚 Verb Dictionary

- Search verbs in **Greek or English**
- View full conjugation tables organized by tense
- Includes pronouns, Greek forms, and English translations

## 🚀 Running Locally

Since the app loads `conjugation.json` via `fetch`, you need a local web server.

### Option 1: Python

```bash
python -m http.server
```

Then open:

```
http://localhost:8000
```
